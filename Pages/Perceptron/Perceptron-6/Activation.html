Activation Function¶

Activation Function on a node is defined as the output that is desired for a given set of input values. There are two types of Activation Functions

  1. Linear Activation Function
  2. Non Linear Activation Function
    • Binary Threshold Unit (BTU)
    • Rectified Linear Neuron
    • Sigmoid
    • Tanhx

Now let us see each one of them in detail.

Linear Activation Function¶

It is defined by the funtion. Realising this activation function is simple but it is computationally limited.

Function: $ \begin{align} y=x \end{align} $

Mapping Domain and Range: $ \begin {align} f:(-\infty,\infty) \mapsto (-\infty,\infty) \end {align} $

Non Linear Activation Function¶

These introduce non linear properties to the neural network. Using a non linear Activation we will be able to generate non-linear mappings from inputs to outputs

1) Binary Threshold Unit:

  • It is also called as the Threshold Logic Unit (TLU).
  • The output is either 0 or 1 hence, it is Unipolar Binary Neuron. (Bipolar Binary Neurons output is -1 or +1 )
  • The problem with this activation function is that even though it is non linear it is discontinuous at 0 and hence it is not differentiable.
  • Hence it cant be used in Gradient descent optimization technique.

Function:

1) Consider $ \begin {align} z=\Sigma x_iw_i \end{align} $, and $ \begin {align} \theta=-b\end{align} $ then

$ y\quad=\quad \begin{cases} 1 & \quad \text{if} \quad z > \theta \\ 0 & \quad \text{otherwise } \end{cases} $

2) Consider $ \begin {align} z=b+\Sigma x_iw_i \end{align} $, then

$ y\quad=\quad \begin{cases} 1 & \quad \text{if} \quad z > 0 \\ 0 & \quad \text{otherwise } \end{cases} $

Mapping Domain and Range: $ \begin {align} f:(-\infty,\infty) \mapsto [0,1] \end {align} $

2) Rectified Linear Unit:

  • This is also popularly called as Relu activation function.
  • It is partly linear and partly non linear.
  • The output is zero for all the negative values.
  • That is the output does not change with the change in the negative values of input.
  • It is simple and is more efficient. It is mainly used in the vanishing gradient problems.

    • </ul> Function:

    Consider $ \begin {align} z=b+\Sigma x_iw_i \end{align} $, then

    $ y\quad=\quad \begin{cases} z & \quad \text{if} \quad z > 0 \\ 0 & \quad \text{otherwise } \end{cases} $

    Mapping Domain and Range: $ \begin {align} f:(-\infty,\infty) \mapsto [0,\infty) \end {align} $

3) Sigmoid:

  • It is fully non linear and is used in many Neural Networks.
  • Used mainly in the binary classification problems.
  • It is also called as the Logistic function.
  • There is no discontinuity and hence derivative can be easily computed.
  • The output is not zero centered.
  • It is observed that in mapping the large range of input is mapped to very small range of output. Hence it makes it very convienient to analize the output.
Function:

Consider $ \begin {align} z=b+\Sigma x_iw_i \end{align} $, then

$\begin {align} y \quad = \frac{1}{(1+{e}^{-x})} \end{align} $

Mapping Domain and Range: $ \begin {align} f:(-\infty,\infty) \mapsto (0,1) \end {align} $

Derivative: $\begin {align} y \quad =\quad y*(1-y)\end{align} $

4) Tanhx:

  • It is fully non linear
  • Output lies in the range of -1 and +1 and is zero centered.
  • As it is zero centered, sometimes it is prefered more than Sigmoid function. But it still faces the Vanishing gradient problem
  • Similar to Sigmoid activation function here also it is observed that in mapping the large range of input is mapped to very small range of output. Hence it makes it very convienient to analize the output.

Function:

Consider $ \begin {align} z=b+\Sigma x_iw_i \end{align} $, then

We know that , $tanh(a \thinspace x) = \Big[\frac{e^{a x}-e^{-a x}}{e^{a x}+e^{-a x}}\Big]$

If $a=1$ in the above equation then we have,

$\begin {align} y \quad = \frac{({e}^{x}-{e}^{-x})}{({e}^{x}+{e}^{-x})} \end{align} $

Mapping Domain and Range: $ \begin {align} f:(-\infty,\infty) \mapsto (-1,1) \end {align} $
